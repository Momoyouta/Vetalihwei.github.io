Einstein学画画

题目描述

Einstein 学起了画画。

此人比较懒~~，他希望用最少的笔画画出一张画……

给定一个无向图，包含 $n$ 个顶点（编号 $1 \sim n$），$m$ 条边，求最少用多少笔可以画出图中所有的边。

输入格式

第一行两个整数 $n, m$。

接下来 $m$ 行，每行两个数 $a, b$（$a \ne b$），表示 $a, b$ 两点之间有一条边相连。

一条边不会被描述多次。

输出格式

一个数，即问题的答案。

样例输入

样例输出

提示

对于 $50 \%$ 的数据，$n \le 50$，$m \le 100$。

对于 $100\%$ 的数据，$1 \le n \le 1000$，$1 \le m \le {10}^5$。

分析

  • 一笔画问题及考察欧拉图及其定理，合理运用该定理即可完成此题
  • 欧拉图部分定理：
    1. 途中所连线段数为奇数的节点(奇点)的数目为0或者2时，该图含有欧拉通路，即可以不重复走某边走过每条边。
    2. 起点为奇点时，终点也为奇点
  • 由此可见，我们只需统计奇点数。每2个奇点所构成的图可以一笔画画完，所以画完整幅图的笔画数为 奇点数/2,不要忘记奇点数为0也可以一笔画画完
